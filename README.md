# SquidServer Installation

This script will help you install the latest **SquidServer AppImage** on your system. 

> [!NOTE]
> The SquidServers application is intended **amd64 (x86_64)** Linux systems.

---

## Installation Commands

You can run the installation script using either `curl` or `wget`. Choose **one** of the methods below to paste into your terminal:

### Method 1: Using cURL
```bash
curl -fsSL https://raw.githubusercontent.com/ourpxi/squidservers-install/refs/heads/main/install.sh | bash
```
### Method 2: Using wget
```bash
wget -qO- https://raw.githubusercontent.com/ourpxi/squidservers-install/refs/heads/main/install.sh | bash
```
